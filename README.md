# Sidney — Personal Portfolio

A modern personal portfolio built with Next.js 14, 
React, TypeScript and Tailwind CSS.

## Tech Stack
- Next.js 14 (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion

## Sections
- About — Profile, role and bio
- Games — Works and project tracking
- Socials — Connect links
- Management — Server management showcase

## Getting Started

Install dependencies:
npm install

Run development server:
npm run dev

Open http://localhost:3000

## Build for Production
npm run build
npm run start

## Deployment
Deployed on Vercel.
